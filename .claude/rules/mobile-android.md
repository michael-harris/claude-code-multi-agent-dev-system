When working with Android code:
- Use Jetpack Compose for new UI (not XML layouts)
- Follow Material Design 3 guidelines
- Use Kotlin coroutines and Flow for async operations
- Use ViewModel with StateFlow for state management
- Use Hilt for dependency injection
- Handle configuration changes properly
- Follow the recommended app architecture (UI -> Domain -> Data)
